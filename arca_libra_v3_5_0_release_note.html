Release Note

LIBRA_World_manager V3.5.0.js

V3.0.0 기준에서 V3.5.0으로 넘어오며 바뀐 핵심 사항들을 보기 좋게 정리한 안내문입니다. 단순 핫픽스 수준이 아니라, 메모리/엔티티/세계관/내러티브/GUI/호환성 전반을 다시 다듬은 확장 업데이트에 가깝습니다.

Scope
대규모 기능 확장
Focus
구조 데이터 정합성 강화
UI / UX
대시보드 · 모바일 · 설정 개선
한 줄 요약
V3.5.0은 기억만 쌓는 플러그인에서 한 단계 더 나아가, 섹션 단위 상태 관리, 세계관 구조화, 분석/재분석 누적 병합, 수동 로어북, 수동 보호, 작업 대시보드까지 갖춘 통합형 장기 컨텍스트 매니저로 확장되었습니다.

1. 메모리 엔진과 턴 처리 강화

  • 턴 리커버리, pending finalize, 세션/채팅 전환 안정성이 보강되었습니다.
  • 메모리 저장, 검색, GC 흐름이 더 안전해졌고, 중복/정리 처리에서 저장 누락 가능성을 줄였습니다.
  • 메모리 검색 가중치의 자동 모드가 더 다양한 장르/상황을 반영하도록 확장되었습니다.

2. 엔티티 / 관계 / 내러티브 관리 고도화

  • 엔티티, 관계, 내러티브에 수동 보호 개념이 추가되어 GUI에서 직접 저장한 정보가 자동 수정에 쉽게 덮어씌워지지 않습니다.
  • GUI에서 보호 설정 / 보호 해제를 직접 전환할 수 있습니다.
  • 엔티티와 내러티브에 재분석 버튼이 추가되어, 현재 채팅 로그 기준으로 다시 구조화할 수 있습니다.
  • 성관념 / 성적 취향 / 직업 / 위치 같은 필드가 더 올바른 카테고리로 정리되도록 분리 처리되었습니다.

3. 세계관 시스템 전면 확장

  • 세계관이 단순 서술 덩어리가 아니라 exists / systems / physics / custom 구조로 더 안정적으로 분배됩니다.
  • 전역 세계 특성 개념이 정리되었고, 복합 세계 신호를 더 폭넓게 자동 감지합니다.
  • 장면용 세계관 보정 기능이 추가되어, 현재 장면에서 어떤 세계관 요소를 우선 해석해야 하는지 보조 프롬프트로 반영합니다.
  • 세계관 재분석 기능이 생겨 현재 채팅 로그 기준으로 세계관만 다시 정리할 수 있습니다.
  • 웹소설 계열 장르 키워드와 퓨전 장르 판정이 강화되어 현대/판타지/헌터/아카데미/선협 등 장르 해석 폭이 넓어졌습니다.

4. Hypa / 과거 대화 분석 흐름 개선

  • Hypa V3, 과거 대화 분석, 과거 대화 재분석 결과가 이제 누적 병합 + 검증 흐름으로 더 보수적으로 통합됩니다.
  • 하이파 반영 시 내러티브, 전역 세계 특성, 장면용 세계관 보정까지 더 폭넓게 같이 갱신됩니다.
  • 동일 Hypa 재가져오기 시 확인창이 떠서 실수로 중복 반영되는 상황을 줄입니다.

5. 수동 로어북 기능 추가

  • 수동 로어북 추가 기능이 생겨, 유저가 직접 작성한 손요약/설정/메모를 `lmai_user_N` 형태로 보관할 수 있습니다.
  • 저장 번호를 직접 지정할 수 있고, 기존 번호 덮어쓰기 시 확인창이 표시됩니다.
  • 원본은 최상위 참고 로어북으로 보존되며, 저장 직후 메인 LLM 분석을 거쳐 LIBRA 구조 데이터에도 병합됩니다.

6. GUI / 대시보드 / 설정 개선

  • 작업 진행 상황을 보여주는 대시보드가 추가되어, 토큰/호출/진행률/백그라운드 작업 상태를 시각적으로 확인할 수 있습니다.
  • 모바일 대응과 위치 조정, 완료 후 자동 닫힘 등 GUI 사용성이 개선되었습니다.
  • 설정은 이제 파일 기반 export/import 방식으로 더 깔끔하게 관리됩니다.
  • 다크 모드 대응, 중앙 정렬, 카드형 정보 표시 등 시각적 완성도가 올라갔습니다.

7. 호환성과 보조 옵션

  • 유저 무시 금지 옵션이 추가되어, 켜면 현재 유저 입력을 최우선 응답 축으로 둡니다.
  • 라이트보드 삽화 태그 호환 옵션이 추가되어, 삽화 모듈의 `<lb-xnai>...</lb-xnai>` 태그를 본문에서 보존할 수 있습니다.
  • 감정 분석, 장르 감지, 자동 가중치 보정도 더 세분화되어 장르문학/웹소설 계열에서의 활용성이 올라갔습니다.

정리

V3.0.0이 기본적인 메모리/구조화 플러그인에 가까웠다면, V3.5.0은 장기 서사 유지, 수동 개입 보호, 분석 결과 검증 병합, 세계관 해석 보조, GUI 가시성까지 포함한 실사용 중심 버전이라고 볼 수 있습니다.
